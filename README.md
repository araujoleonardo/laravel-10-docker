## Setup docker para aplicação laravel + vuejs


### Exemplo de configuração para o .env

```
APP_URL=http://localhost

DB_CONNECTION=mysql
DB_HOST=mysql
DB_PORT=3306
DB_DATABASE=db-name
DB_USERNAME=admin
DB_PASSWORD=admin
DB_ROOT_PASSWORD=root

CACHE_DRIVER=redis
QUEUE_CONNECTION=redis
SESSION_DRIVER=redis

REDIS_HOST=redis
REDIS_PASSWORD=null
REDIS_PORT=6379
```
